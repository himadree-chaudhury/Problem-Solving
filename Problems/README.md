# Important STL Functions

### Vector
Vector is a dynamic array that can resize itself automatically when elements are added or removed. It provides fast access to elements and allows for efficient insertion and deletion at the end of the vector.

### String
String is a sequence of characters that can be manipulated and stored in a container. It provides various functions for string manipulation, such as concatenation, substring extraction, and searching for specific characters or substrings.

### Pair
Pair is a simple container that holds two values of different types. It is often used to store related data together, such as a key-value pair or a coordinate point.

### Map
Map is a container that stores key-value pairs. It allows for fast retrieval of values based on their corresponding keys. The keys in a map are unique, and the values can be accessed using the keys.

### Tuple
Tuple is a container that can hold a fixed number of elements of different types. It allows for grouping related data together, such as a person's name, age, and address. Tuples are often used when you want to return multiple values from a function or when you want to store heterogeneous data in a single container.

### Set
Set is a container that stores unique elements in a sorted manner. It provides fast access to elements and allows for efficient insertion and deletion. Sets are often used to store a collection of unique items, such as a list of unique names or a set of unique numbers.

### Stack
Stack is a container that follows the Last In First Out (LIFO) principle. It allows for efficient insertion and deletion of elements at the top of the stack. Stacks are often used in algorithms that require backtracking or undo functionality.

### Queue
Queue is a container that follows the First In First Out (FIFO) principle. It allows for efficient insertion of elements at the back of the queue and deletion of elements from the front of the queue. Queues are often used in algorithms that require processing items in a specific order, such as breadth-first search or task scheduling.

### Deque
Deque (double-ended queue) is a container that allows for efficient insertion and deletion of elements at both the front and back of the queue. It provides fast access to elements and can be used as both a stack and a queue. Deques are often used in algorithms that require flexible data structures, such as sliding window problems or implementing a circular buffer.

### Tree
Tree is a hierarchical data structure that consists of nodes connected by edges. It is often used to represent relationships between data, such as a family tree or a file system. Trees provide efficient searching, insertion, and deletion operations, making them useful for various applications, such as binary search trees, AVL trees, and B-trees.
